Desenvolvimento de aplicações web com o framework ReactJs e Bootstrap. Durante o desenvolvimento, estou sempre atenta às melhores práticas para seguir um alto nível de padrões de código e no desenvolvimento com testes. Também ajudo desenvolvedores menos experientes no dia-a-dia a evoluir tecnicamente juntos e entregar um projeto altamente escalável.

Contribuição com tecnologias do frontend e trabalho na reestruturação de projetos antigos para uma arquitetura de frontend escalável. Quando necessário, criadas interfaces sem depender de um UI designer. Experiência em UI e UX para garantir melhor usabilidade. Otimização WebSites e interfaces. Criação de Designer Responsivo. Criação de Ferramentas e elementos para interação e efeitos visuais.

Tratamento de questões de segurança, performance, UI/UX, arquitetura de aplicações, modularização sempre utilizando boas práticas de desenvolvimento e metodologias ágeis. Lidando também com os principais tipos de testes como: testes automatizados, testes unitários, integração. Experiência em Testing e Debugging e otimização de rendimento.

Criação de dashboards DAX e relatórios em Power BI para interagir com as opções retiradas do banco de dados. Criação e Validação de Formulários
